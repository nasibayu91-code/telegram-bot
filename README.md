## ⚙️ Настройка

### 1. Получи токен бота
Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен

### 2. Узнай свой Telegram ID
Напиши [@userinfobot](https://t.me/userinfobot)

### 3. Отредактируй `config.py`
```python
BOT_TOKEN    = "123456:ABC-xyz..."   # токен от BotFather
ADMIN_ID     = 123456789             # твой Telegram ID
PAYMENT_DETAILS = "USDT TRC-20: Txx...твой_адрес"
```

### 4. Установи зависимости
```bash
pip install -r requirements.txt
```

### 5. Запусти бота
```bash
python bot.py
```

## 🚀 Деплой на Railway (бесплатно 24/7)

1. Зарегистрируйся на [railway.app](https://railway.app)
2. Создай новый проект → **Deploy from GitHub**
3. Загрузи все файлы на GitHub (или используй Railway CLI)
4. В настройках проекта добавь переменную окружения:
   - `BOT_TOKEN` = твой токен
5. В `config.py` замени значения или используй `os.environ`:
   ```python
   import os
   BOT_TOKEN = os.environ.get("BOT_TOKEN", "YOUR_TOKEN")
   ADMIN_ID  = int(os.environ.get("ADMIN_ID", "123456789"))
   ```
6. Railway автоматически запустит `Procfile`

## 💳 Платёжные системы

### ⭐ Telegram Stars
- Встроенная оплата Telegram, работает сразу
- Настрой в BotFather: `/mybots` → выбери бота → `Payments`
- Курс: 1 Star = 10 токенов (меняется в `config.py`)

### 💵 USD вручную
- Игрок вводит сумму → получает реквизиты → отправляет чек
- Ты получаешь уведомление с кнопками "Одобрить / Отклонить"
- Укажи свои реквизиты в `PAYMENT_DETAILS`

## ❓ Частые вопросы

**Q: Бот падает при старте?**
A: Проверь `BOT_TOKEN` и `ADMIN_ID` в `config.py`

**Q: Stars не работают?**
A: Включи платежи у BotFather: `/mybots` → бот → `Payments` → Telegram Stars

**Q: Как изменить шанс выигрыша?**
A: В `config.py` измени `WIN_RATE = 0.30` (0.30 = 30%)
